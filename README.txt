В игру можно сыграть в двую режимах - с классическими настройками морского боя(поле 10на10, 4 однопалубных, 3 двухпалубных, 2 трехпалубных, один четырехпалубный) и введя конфигурацию самостоятельно. При выборе самостоятельного ввода пользователь должен ввести размер поля(от 6 до 30 в каждом измерении) и количество корабелй каждого типа (неотрицательное число).
Далее, если не получилось сгенерировать поле по настройкам, заданным пользователем ему будет предложено ввести конфигурацию заново.
После начала игры пользователь должне провести N выстрелов, чтобы убить все корабли. Чем меньше выстрелов, тем результат лучше. Для выстрела требуется ввести в одной строке 2 числа x y, означающих координаты точки, в которю будет произведен выстрел. Диапозон значений - от 1 до максимальной размерности измерения.
Полсе каждого выстрела пользователь увидит обновленное поле боя, на котором "o" обозначает промах, "*" обозначает подбитый корабль, "." обозначает неоткрытую клетку.
Когда пользователь уничтожит все корабли, программа выведет ему количество совершенных выстрелов.
